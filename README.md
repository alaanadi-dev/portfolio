# Alaa Nady Sayed — Portfolio

My personal portfolio showcasing my Flutter development projects, skills, and professional experience.

**Live site:** https://alaanadi-dev.github.io/portfolio/

## Stack

Pure HTML, CSS, and vanilla JavaScript — no build step, no framework.

- Dark/light theme toggle with `localStorage` persistence
- Scroll-triggered reveal animations via `IntersectionObserver`
- Animated stat counters and skill progress bars
- Typing animation, floating decorative icons, glassmorphism UI
- Responsive layout, semantic HTML, SEO meta + JSON-LD

## Structure

```
index.html
assets/
  css/style.css
  js/main.js
  img/            → app screenshots + headshot
  cv/             → downloadable CV (PDF)
```

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static file server.
